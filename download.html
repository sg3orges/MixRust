Télécharger AudioRust

Téléchargez la dernière version de AudioRust, un lecteur audio développé en Rust avec une interface graphique GTK et un égaliseur permettant de modifier les basses et les aigus en temps réel.

Version actuelle

Version : v1.0

Date : 2026

Télécharger

Plateformes supportées

  • Windows
  • Linux
  • MacOS (expérimental)

Installation

Si vous souhaitez compiler le projet vous-même :

git clone https://github.com/TON-REPO
cd projet
cargo build --release
